# EXPENSE TRACKER
Expense tracker is basically designed to give users a total of all expenses made.
A basic expense tracker with an interactive menu;
- Adding a new expense
- Viewing current expenses in memory
- Calculating the total expense amount
- A placeholder for delete functionality
- Exiting the program

How it works:
- Expenses are stored in a Python list named `expense`.
- When a new expense is added, the entire list is written to `expenses.csv`.
- The CSV file contains two columns: `name` and `amount`.

Notes:
- Delete functionality is not implemented yet.
- Data only persists in `expenses.csv` while the program is running and after items are added.
